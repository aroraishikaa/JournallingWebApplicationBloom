<h1>Journalling Web Application - Bloom</h1>

<h2>Description</h2>
Bloom is a user-friendly journaling application that offers individuals a seamless platform to engage with their thoughts and emotions through daily journaling. The mission is to help users build a sustainable journaling habit, promoting personal growth and self-reflection. Bloom allows users to effortlessly create, view, and reflect on their digital journal entries, fostering both creativity and introspection. Link to walk through of web application features and design: https://www.dropbox.com/scl/fi/wruirkfgqttxr46a0tdep/Video-Walkthrough-of-Bloom.mp4?rlkey=idcw6jisqr97dhtwycbuvm2ka&st=bhs4rssc&dl=0.
